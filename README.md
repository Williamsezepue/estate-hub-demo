# Estate Hub — Real Estate Marketing Demo

A polished real estate marketing demo built with **Next.js** and **Tailwind CSS**.  
Estate Hub showcases property listings, lead capture forms, and responsive layouts tailored for real estate businesses.  
Designed for performance, accessibility, and clarity to help attract and convert clients.

---

## Features

- Property showcase with dynamic image galleries
- Lead capture forms integrated with backend services
- Responsive layouts optimized for mobile and desktop
- Modern animations and transitions for engaging user experience
- Accessibility‑first design with SEO‑friendly structure

---

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Supabase](https://supabase.com/)
